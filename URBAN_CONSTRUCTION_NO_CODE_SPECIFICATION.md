**Формула расчёта статуса:**
```
Отклонение % = (Факт - План) / План * 100

IF Факт <= План:
  Статус = ✅ Зелёный (Экономия или в плане)
ELSE IF План < Факт <= План * 1.10:
  Статус = 🟡 Жёлтый (Перерасход до 10%)
ELSE:
  Статус = 🔴 Красный (Критический перерасход)
```

**Интерактивность:**
- Hover на столбец → Tooltip: "План: XXX млн ₽ | Факт: XXX млн ₽ | Отклонение: +XX%"
- Клик на подрядчика (внизу) → Drawer справа с историей платежей

## 🧮 ФОРМУЛЫ KPI (автоматические расчёты)

### Бюджет
```
variance = actualSpent - plannedAmount
percentage = (actualSpent / plannedAmount) * 100

status = 
  IF actualSpent <= plannedAmount: "on_track" 
  ELSE IF actualSpent <= plannedAmount * 1.10: "overrun_minor"
  ELSE: "overrun_critical"
```

### Производство
```
projectProgress = SUM(sectionProgress * sectionWeight) / 100
criticalPathShift = MAX(plannedDate - actualDate) FOR all isOnCriticalPath = TRUE
workerShortage = plannedWorkers - actualWorkers
```

### Продажи
```
funnelConversion = stageN / stageN-1 * 100
averagePricePerM2 = totalRevenue / totalSoldArea
```

### Снабжение
```
financialCompletion = SUM(paidPercent * itemValue) / SUM(itemValue)
physicalCompletion = SUM(deliveredPercent * itemValue) / SUM(itemValue)
criticalDelays = COUNT(WHERE delayDays > 0)
```

### 20/80 Парето
```
priorityScore = (impactDays * 0.5) + (financialWeight * 0.3) + (blockedTasksCount * 0.2)
topTasks = SORT BY priorityScore DESC LIMIT 7
```

### Просрочка
```
delayDays = MAX(0, TODAY - plannedDeadline)

classification =
  IF isOnCriticalPath OR delayDays > 7 OR risk > 1M: "CRITICAL"
  ELSE IF 3 <= delayDays <= 10: "WARNING"  
  ELSE: "MINOR"
```
